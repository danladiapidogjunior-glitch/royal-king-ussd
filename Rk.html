Important Rules

Recipient cooldown is currently 60 minute(s). A new order for the same recipient is blocked only when the previous order is still On Hold or Processing inside the cooldown time. Completed orders are accepted immediately. Once cooldown expires, the next order is accepted even if the old order is still On Hold/Processing.

Order Status Check Endpoint

Use this endpoint to let your own website/app poll and auto-update your customer’s order status.

POSThttps://rkdeals.cloud/wp-json/rkapi/v1/order-status

Alias also supported:

https://rkdeals.cloud/wp-json/rkapi/v1/check-transaction-status

Request Body

{
  "transaction_id": "rkapi_12345"
}

Or:

{
  "order_id": 12345
}

Success Response

{
  "success": true,
  "transaction_id": "rkapi_12345",
  "order_id": 12345,

JavaScript Status Polling Example

async function checkStatus(transactionId) {
  const res = await fetch("https://rkdeals.cloud/wp-json/rkapi/v1/order-status", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "rk-api-token": "YOUR_API_TOKEN"
    },
    body: JSON.stringify({ transaction_id: transactionId })
  });
  const data = await res.json();
  return data.status;
}
